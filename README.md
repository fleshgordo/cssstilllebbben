# cssstilllebbben

Auf der Website siehst du ein Bild mit mehreren Alltagsobjekten, angeordnet auf einem Raster. Deine Aufgabe: baue das Bild **ausschliesslich mit CSS** nach – keine Bilder, keine SVGs, nur reines CSS. Alles ist erlaubt: `grid`, `clip-path`, `gradients`, `box-shadow`, Pseudo-Elemente, was auch immer.

Ziel: die Grenzen von CSS ausloten & Web inspector kennenlernen

Fertig? Screenshot machen und aufs Bild ziehen – die Zeit stoppt.

Advanced challenge (nachdem der Time stoppt ;)): Mache das Grid responsive (wie sieht es auf Mobile aus). Untersuche die Möglichkeiten von Grid-Animationen. Wie viel geht da?
